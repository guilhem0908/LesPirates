# Les Pirates

Ce projet est un jeu de cartes en console opposant deux pirates.  
Chaque joueur commence avec 5 cœurs (points de vie) et doit atteindre 5 points de popularité pour gagner, ou survivre plus longtemps que son adversaire.

## Structure du projet

- **src/**  
  Contient le code source, divisé en :
  - **app/** : Point d'entrée de l'application.
  - **controller/** : Gestion des interactions entre la logique métier et la vue.
  - **model/** : La logique métier (classes Joueur, Carte, etc.).
  - **view/** : L’affichage en console.

- **test/**  
  Contient les tests unitaires.

- **docs/**  
  Documentation et rapports.

- **UML/**  
  Diagrammes UML du projet.

## Instructions

1. Cloner le dépôt Git.
2. Compiler le projet (ex. avec Maven ou Gradle).
3. Lancer la classe `Main` dans le package `app` pour démarrer le jeu.
